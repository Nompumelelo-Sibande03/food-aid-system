Food Aid System: Community Connect

Overview

This project is a system designed to connect food donors with individuals and organizations in need, helping reduce food waste and improve food distribution.

Problem

Large amounts of food are wasted while many communities experience food shortages.

Solution

The system allows donors to post available food and enables receivers to request and collect it efficiently.

Users

* Donators
* Receivers
* Admin

 Process Flow

1. Donator uploads food
2. System notifies receivers
3. Receiver requests food
4. Admin verifies request
5. Food is collected

Database Design

Includes tables such as:

* Users
* Donations
* Requests

Security Considerations

* User authentication required
* Admin approval for requests
* Data protection for users

Technologies Used

* SQL Database
* System Design Tools (Draw.io)
* VS Studio

Key Learnings

* System design and user interaction
* Database structuring
* Importance of secure and efficient systems
